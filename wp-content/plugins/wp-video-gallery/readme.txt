== Description ==

WP Video Gallery is the Powerful WordPress Plugin. You can easily manage your videos and Images of Youtube, Vimeo, Dailymotion, Twitpic, and Instagram (Photos).
You can create your gallery of your list of Video or Photo. Easily control it from Backend.
WP Video Gallery has light box Effect to show your video quickly and Responsive. WP Video Gallery is Bright Elegant and have Multiples skins.

== Installation ==

Installation of WP Video Gallery Plugin is Easy!.

1. Upload `wp-video-gallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php if( function_exists('wpvg_get_all_categories') ) wpvg_get_all_categories(); ?>` in your templates or use shortcode [video_categories]

4.That's it. You're ready to go!

Minimal Requirements
    PHP: 5.x.x
    mySQL: 4.0, 4.1 or 5.x
    WordPress: 3.4 or newer

Recommended Requirements
    PHP: 5.2.x or newer
    mySQL: 5.x
    WordPress: 3.9 or newer


== Frequently Asked Questions ==

= How to Display Individual  Gallery? =
You Can Get Individual Video Gallery in two ways.
1. Use Shortcode with ID [gallery_videos id="1"] to your WordPress Page or Post
2. Place the Code with your Gallery ID <?php if( function_exists('wpvg_get_category_videos') ) wpvg_get_category_videos('1'); ?> in Template Pages. like index.php or single.php

= How to Get All Videos? =
You can get Video Gallery in two ways.
1. Use Shortcode [video_categories] to your WordPress Page or Post.
2. Place the Code <?php if( function_exists('wpvg_get_all_categories') ) wpvg_get_all_categories(); ?> in Template Pages. like index.php or single.php

= Which videos URL i can use? =
Currently WP Video Gallery Supported the Following Video Media's
